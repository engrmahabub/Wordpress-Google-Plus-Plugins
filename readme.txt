== Description ==
The Google+ Plugins is a wordpress widget plugin. It lets you easily embed and promote any Google Plus Page or User profile follow button on your website. Just like on Google Plus, your visitors can follow the Page or user without leaving your site.

= Features =
* Show or hide follow count bubble.
* Show horizontal or vertical bubble.
* Bubble size change. You can set bubble as small, medium or large.
* Widget title change.
* You can set page or profile follow button by writing its valid url.

== Installation ==
1. Upload the entire `sm-google-plus` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now SM Google Plus widget abailable in your widget section.

= Usage =

1. Go to Appearance > Widgets.
2. Enable SM Google+ Follow widget.
3. Enter widget title
4. Enter Google+ user or page url;
5. Check of uncheck options.
6. Hit the save button.
